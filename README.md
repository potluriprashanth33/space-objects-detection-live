# 🚀 SPACED: Space Object Detection System

🌌 To Space and Back, Safely

An AI-powered Space Object Detection System designed to simulate real-time safety monitoring for space travel 🚀.
This project combines computer vision, simulated datasets, and interactive web UI to demonstrate how AI can detect and classify space-related objects.

# 🧠 Project Overview

This project is built to simulate how AI-powered safety systems can be used in future space missions.

Using Duality AI simulated datasets, the model is trained to detect objects in a space-like environment, ensuring reliability even without real-world data.

# ⚙️ Workflow (How It Works)

🧩 Step-by-Step Pipeline

1️⃣ Data Collection

Simulated images generated using Duality AI

Covers various space objects and scenarios


2️⃣ Model Training

AI model trained on synthetic datasets

Learns object patterns in space environments


3️⃣ Frontend Interface

Built using HTML, CSS, JavaScript

Anime.js used for smooth animations


4️⃣ Detection Process

🎥 Live camera OR image upload

🧠 Model processes input

📦 Objects detected & classified

📊 Results displayed with confidence


5️⃣ User Interaction

Toggle sound alerts 🔊

Fullscreen detection mode ⛶

Real-time stats & detection history
